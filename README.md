## Running the solutions

For running the code you need the [Deno runtime](https://deno.land/) installed.
Run the code via

```bash
deno run --allow-read 01/a.ts
# or
deno test 01/a.ts --allow-read
# or
npm run test
# or
npm run test:watch
```
